Note di installazione
Encoder v1.0

Requisiti di sistema

Dal punto di vista hardware, Encoder non necessita di specifiche particolari. La questione cambia dal punto di vista software in quanto Encoder è un programma scritto in linguaggio Java. Per questo motivo ha bisogno di strumenti esterni sia per la compilazione che per l'esecuzione in ciascune delle principali piattaforme operative (Linux, Microsoft Windows e MacOs).

Compilazione ed esecuzione

Dopo avere spacchettato la distribuzione, per compilare i sorgenti o eseguire il programma occorre scaricare ed installare Java SE Development Kit, indipendentemente dal sistema operativo. Download e altre documentazioni sono disponibili qui. Su Linux (base Debian), in alternativa, è possibile semplicemente eseguire da terminale il comando "sudo apt-get install openjdk-11-jre". Per la compilazione, bisogna aprire da terminale la cartella encoder10\src\frame1 della distribuzione ed eseguire il comando "javac *.java". Encoder non ha bisogno di installazione, ma va semplicemente eseguito. Per la sua esecuzione si possono intraprendere due strade:

  1. Si può utilizzare il file eseguibile Encoder.jar, posizionato nella cartella encoder10\bin della distribuzione. Su Windows basta fare un doppio click sinistro, mentre su Linux o MacOs da terminale occorre posizionarsi nel percorso dove si trova il file JAR ed usare il comando "java -jar Encoder.jar";
  2. Oppure, aprendo il terminale nella cartella encoder10\bin, può essere usato il comando "java -classpath . frame1.Interfaccia".
